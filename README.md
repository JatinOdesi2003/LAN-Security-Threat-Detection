# 🛡️ LAN Security & Network Threat Detection

A practical setup and analysis framework for monitoring local area network (LAN) traffic, identifying vulnerabilities, and detecting intrusion attempts.

## 🚀 Key Features
- **Packet Inspection:** Live network packet capturing and deep packet analysis using Wireshark.
- **Intrusion Detection (IDS):** Custom rule generation and alert systems using Snort IDS.
- **Network Defense:** Perimeter security configuration and traffic filtering rules via PfSense.
- **Threat Logging:** Analysis of suspicious network traffic and port scanning detection.

## 🛠️ Tools & Technologies
- Wireshark
- Snort IDS
- PfSense / Firewall Rules
- TCP/IP Protocols & Wi-Fi Security
